## 0. Tek cümlelik konum
**"Öğrenci koçluğunu tahminden çıkarıp, ölçülebilir başarıya dönüştüren güvenilir dijital sistem."**

İki söz her şeyi taşır:
- **BAŞARI** = ölçülebilir sonuç (net gelişimi, program tamamlama, hedefe ulaşma, geride kalmadan uyarı).
- **GÜVEN** = şeffaflık (veli görür) + gizlilik (KVKK, koça-özel notlar) + kurumsallık + sıfır-risk deneme.

## 1. GÜVEN — 7 sütun (her temas noktasında inşa edilir)
1. **Görsel tutarlılık**: aynı petrol(#0e7490)+altın(#f1b422)+beyaz, aynı amblem/wordmark — web, store, splash, ikon, ekran görselleri. Tutarlılık = profesyonellik = güven.
2. **Şeffaflık**: "Veli her şeyi görür" yerleşik vaat. Gizlenecek bir şey olmayan sistem güven verir.
3. **Gizlilik**: "KVKK'ya uygun · veriler şifreli · koça-özel notlar yalnız koçta" — açıklamada, görselde (#8 güven kompoziti), uygulama-içi.
4. **Kanıtlanabilir sonuç**: net trendi grafikleri + tamamlama % + "geçen haftaya göre +X" → vaadi kanıtla.
5. **Sosyal kanıt**: ilk 2 hafta memnun koç/veliye nazik 5-yıldız isteği (StoreReview API). **Sahte yorum YASAK** — gerçek 10-15 beş yıldız algıyı kurar.
6. **Kurumsal kimlik**: her yerde "ETÜTKOÇ Rotam". Satıcı adı ileride ETÜTKOÇ (Organization). Destek e-postası + web + KVKK sayfası canlı.
7. **Sıfır-risk**: "14 gün ücretsiz dene, kart yok". Risk-tersine-çevirme = dönüşüm.

## 3. HEDEF KİTLE HUNİSİ — KOÇ-ÖNCELİKLİ
**Çekirdek karar: ödeyen kitle = bağımsız koç.** Pazarlama bütçesi/mesaj sırası koça odaklanır; veli/öğrenci/kurum koç üzerinden viral gelir.

```
Bağımsız Koç (birincil, hızlı dönüşen, ödeyen)
   └─→ davet ettiği Öğrenciler + Veliler (organik, viral, ücretsiz büyüme)
Kurum (ikincil, yüksek değer, uzun döngü → iletişim formu/satış)
```

- **Neden koç**: karar verici + ödeyen + her koç ortalama N öğrenci+veli getirir (yerleşik büyüme döngüsü).
- **Kurum**: self-signup YOK → /pricing iletişim formu → süper admin onboarding (yüksek dokunuş, yüksek LTV).

## 4. LANSMAN SIRASI (kritik yol)
**Faz 0 — Kapalı test (ZORUNLU, yeni kişisel hesap):** 12 test kullanıcısı / 14 gün.
  - Test kullanıcıları: mevcut gerçek koçlar + güvenilen veliler (Google grup veya e-posta listesi). Gerçek kullanım = gerçek geri bildirim + ilk yorumlar.
**Faz 1 — Soft launch:** üretim erişimi açılınca mevcut koç tabanına duyuru (e-posta + uygulama-içi). "Artık cebinizde."
**Faz 2 — Organik/ASO:** store optimizasyonu olgunlaşır; "öğrenci koçu", "deneme takibi", "lgs/yks" arama trafiği.
**Faz 3 — Büyüme döngüsü:** koç→davet, veli-memnuniyeti→5 yıldız, referral.

## 6. BÜYÜME DÖNGÜLERİ (yerleşik, ücretsiz)
1. **Koç → kullanıcı**: her koç öğrenci+veli davet eder → kullanıcı tabanı koç sayısıyla çarpan etkisiyle büyür.
2. **Veli memnuniyeti → 5 yıldız → ASO**: şeffaflık velide güven → yorum → arama sıralaması.
3. **Web ↔ store tutarlılığı**: anasayfa + /pricing + store aynı vaat/görsel → her kanal diğerini güçlendirir.
4. **Net gelişimi = paylaşılabilir başarı**: veli "çocuğum +19 net" hikayesini paylaşır (organik).

## 8. BAŞARI METRİKLERİ (ne ölçülür)
- **Edinme**: indirme, store sayfa→indirme dönüşümü (görsellerin etkisi).
- **Aktivasyon**: kayıt → ilk öğrenci ekleme → ilk program (koç "aha" anı).
- **Gelir**: trial → paid dönüşümü, koç-başı LTV.
- **Tutundurma**: 7/30 gün retention, haftalık aktif koç.
- **Güven**: ortalama rating (≥4.5 hedef), yorum kalitesi.
- **Viral**: koç-başı davet edilen öğrenci/veli sayısı.

## 9. KIRMIZI ÇİZGİLER (güveni korur)

- Abartılı/yanlış vaat YOK (gerçek özellik = gerçek vaat).
