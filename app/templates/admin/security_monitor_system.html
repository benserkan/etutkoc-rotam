🔧 Sistem Hata Kamerası

Programın hangi sayfasında, kim için, ne hatasıyla çöktüğünü gösterir. Aynı bug birden çok kullanıcıyı vurmuş olsa bile tek satırda toplanır ("27 kez oldu" gibi).

{# "Bu sayfada ne yazıyor?" — sade açıklama #}
💡 Bu sayfada ne yazıyor? (terim açıklamaları)
Hata grubu — aynı bug farklı kullanıcıları vurursa, hepsi tek satırda toplanır (örn. "şu sayfada bu hata 27 kez oldu"). Karışıklık olmasın diye.
Açık / Çözüldü — bug fix yapıp yayına alınca "çözüldü" işaretle. Daha sonra aynı hata tekrar oluşursa otomatik açılır ("bug geri geldi" sinyali).
500 hatası — sunucu tarafında bir şey çöktü, kullanıcının suçu yok. Olduğu gibi buraya kaydedilir. (Teknik: HTTP 5xx hataları)
İstisna / Exception — Python kodunun beklemediği bir durumla karşılaşması (örn. "sayı yerine harf geldi", "bağlantı kesildi"). Yakalanmazsa sayfa çöker.
Stack trace — hatanın hangi kod satırından kaynaklandığını gösteren teknik döküm. Geliştiriciye lazım, bug'ı bulmak için.
İmza — aynı bug'ı tanımak için kullanılan benzersiz parmak izi (sayfa + hata tipi + kod konumu). Sayesinde dedup yapılır. (Teknik: signature)
